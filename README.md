# System Monitor

<img width="895" height="588" alt="image" src="https://github.com/user-attachments/assets/096044b6-d0b2-4cc4-aa57-62235a233d59" />

System Monitor — это лёгкое и удобное приложение для мониторинга системных ресурсов Windows. Оно показывает загрузку процессора, оперативной памяти, дисков реальном времени, имеет современный интерфейс и работает максимально быстро.

Приложение создано на WPF с акцентом на стабильность, плавность и минимальное потребление ресурсов. Интерфейс выполнен в стиле Fluent Design, поддерживает тёмную тему и аккуратную кастомную иконку.

## Возможности

- Мониторинг CPU, RAM, дисков в реальном времени  
- Современный интерфейс с плавной анимацией  
- Поддержка тёмной темы  
- Минимальная нагрузка на систему  
- Удобный MSI‑установщик с ярлыками  
- Автоматическое обновление данных без задержек  

## Установка

Скачайте установщик из раздела Releases и запустите его. После установки приложение появится в меню «Пуск» и на рабочем столе.

## Требования

Windows 10 или Windows 11 (x64).  
Дополнительные компоненты не требуются.
